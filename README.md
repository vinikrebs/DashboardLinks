## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e Github
- Figma

## 💻 Projeto

A lista de Links é um projeto que apresenta os seus principais links de acesso do seu portifólio e redes sociais.
